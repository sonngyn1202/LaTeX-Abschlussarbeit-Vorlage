# LaTeX-Abschlussarbeit-Vorlage
Vorlage zur Erstellung einer Abschlussarbeit über eine IDE wie z.B. VSCode.
Die Vorlage wurde erstellt nach den Formalien der HSWT (Stand 2026).

# Welche Programme braucht man?
siehe https://www.latex-project.org/get/

## Windows
- MiKTeX (oder TeX Live)
- Strawberry Perl – wird von MiKTeX für manche Pakete benötigt
- VS Code + LaTeX Workshop (VS Code Erweiterung)

## Mac
- MacTeX – alles inklusive (Perl ist auf macOS bereits vorinstalliert)
- VS Code + LaTeX Workshop (VS Code Erweiterung)

# Was wird in der Anleitung erklärt?
- Grafiken einfügen
- zitieren
- Tabellen erstellen
- Deckblatt
- Formalien
- Formeln
- Verzeichnisse
- Abkürzungen
